# Image assets — what to swap in before launch

Every image here is a **branded placeholder SVG** generated by `scripts/gen-placeholders.mjs`
(navy/blue gradient + truck motif + label). Replace them with real photography/artwork using the
**same file paths** and the site picks them up automatically — no code changes needed.

Prefer optimized `.webp`/`.avif` or high-quality `.jpg`. If you change file extensions, update the
matching `image:`/`cover:`/`src:` path in the relevant `content/*.ts` file.

## Brand / logo

- The navbar/footer logo is currently a **CSS + inline-SVG recreation** in
  `components/layout/Logo.tsx`. Drop the official logo (SVG preferred) into `public/` and replace the
  `<Logo />` internals, or swap to a `next/image` of the real file. A favicon/app icon set is also
  recommended (`app/icon.png`, `app/apple-icon.png`).
